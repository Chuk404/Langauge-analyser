Language Analyser

A cultural slang translator and explainer powered by the OpenRouter AI API. 
Type in slang or informal language and get a full translation alongside a detailed explanation of the cultural context and meaning behind it.

About

Language Analyser was built to bridge the gap between slang and understanding. 
Whether you've come across unfamiliar slang online or want to understand the cultural context behind a phrase,
Language Analyser translates it into plain language and explains what it actually means and where it comes from.

Features

Translates slang and informal language into plain English
Provides a separate explanation of the cultural context and meaning
Powered by the OpenRouter AI API for accurate, context-aware responses
Simple, clean web interface

Tech Stack

Backend: Java, Spring Boot
Frontend: JavaScript, CSS, HTML
AI: OpenRouter API
Version Control: GitHub

Getting Started

Prerequisites
Java installed on your machine
A terminal (VS Code terminal, Command Prompt, or any terminal)

Installation


Clone the repository

Open the project in VS Code or your preferred editor
Open a Terminal  or press Ctrl + Shift + '

To Start the Spring Boot server
type cd backend
then type .\mvnw.cmd spring-boot:run

Note: You must run the server before using the site, otherwise you will get a "server not found" error.

Open the frontend index.html file in your browser

Usage

Make sure the backend server is running (see above)
Open the site in your browser
Type a word or phrase containing slang into the input box
Get back a full translation and a cultural explanation of what it means
